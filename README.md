# mpcs-57200-final
Dataset Analyzer


Begin by installing dependencies and setting up venv:

1. python3 -m venv venv

2. source venv/bin/activate

3. pip install --upgrade pip

4. pip install -r requirements.txt

To run the app:
5. streamlit run dataset_analyzer.py
